# Antigravity Premium Dashboard

A modern, highly aesthetic web application built with a responsive glassmorphic dashboard layout, CSS Custom Properties, curated dark/light color themes, ambient glows, dynamic mouse-parallax glows, and responsive components.

## Features

- **Rich Aesthetics**: High-end glassmorphic UI cards, subtle transitions, and custom HSL color systems.
- **Dynamic Ambient Glows**: Dual background glow elements that move subtly based on the user's cursor position.
- **Theme Switcher**: Instant dark and light theme toggling with modern transitions, automatically saved to `localStorage`.
- **Responsive Layout**: Perfect scaling from 4K displays down to mobile screens.
- **Interactive Micro-animations**: Toggling checklists, dynamic chart bars clicking, notification dismissal, search handlers, and dynamic statistics increments.

Simply open `index.html` in your favorite web browser to see it in action, or deploy it instantly using any web server!
